Application Documentation
=========================
Braindump is a simple, free, and open note platform that can help you organize your life.

Getting Started
---------------
There are two ways to get started with braindump. You can either sign up for a free account on the `main braindump website`_ or you can run braindump yourself on your own server. This section covers using the public version of braindump. If you want to learn more about self hosting braindump please see the documentation regarding :doc:`/hosting`.
